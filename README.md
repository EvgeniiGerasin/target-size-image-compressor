# icompressor
<img width="500" height="350" alt="image" src="https://github.com/user-attachments/assets/9487386e-af70-44db-a709-29c457928b78" />

Веб-инструмент для сжатия изображений до заданного размера. Доступен по адресу: https://icompressor.vercel.app/

## Описание

Позволяет выбрать несколько изображений и сжать их до заданного размера, с возможностью скачивания всех файлов в ZIP-архиве.

<img width="500" height="350" alt="image" src="https://github.com/user-attachments/assets/bf916fab-0104-4003-9479-b699ff527737" />

## Особенности

- Сжатие до заданного размера (в килобайтах)
- Поддержка множественного выбора файлов
- Предварительный просмотр изображений
- Загрузка всех сжатых изображений в одном ZIP-архиве
- Совместимость с популярными форматами изображений
